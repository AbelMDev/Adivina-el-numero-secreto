# Adivina-el-numero-secreto

## Indicaciones:
Descripción:
El programa debe generar un número secreto entre 1 y 20.
Los jugadores (o el grupo) intentan adivinar el número.
Después de cada intento, el programa les dice si el número secreto es mayor o menor que el número ingresado.
El juego termina cuando el número se adivina correctamente.


Extensiones:

1. Límites de intentos: permitir solo 5 intentos antes de perder.

2. Repetir juego: preguntar al final si quieren jugar otra vez.

3. Niveles de dificultad: (fácil: 1–10, medio: 1–20, difícil: 1–50).
